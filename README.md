# 🏥 AI Medical Diagnosis System

An intelligent web-based medical diagnosis system that predicts diseases based on user symptoms using **Artificial Intelligence techniques** such as **Artificial Neural Networks (ANN)** and **Semantic Networks**.

🔗 Live Demo: https://yingtao.pythonanywhere.com  
📂 GitHub Repo: https://github.com/kukkamallaramya599-del/Medical-System

---

## 📌 Project Overview

The AI Medical Diagnosis System helps users identify possible diseases based on selected symptoms.  
It integrates:

- Rule-based diagnosis
- ANN-based prediction
- Semantic Network visualization

This project demonstrates the combination of **symbolic AI** and **machine learning** techniques.

✅ User Registration & Login System

## 🧩 System Architecture

1. User selects symptoms  
2. System processes input  
3. ANN predicts disease probability  
4. Rule-based system calculates match %  
5. Best disease is selected  
6. Doctor recommendation is provided  
7. Semantic network visualizes relationships
